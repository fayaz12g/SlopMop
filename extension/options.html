How to get your Gemini API key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Get API key" or "Create API key"
  4. Copy the generated key and paste it below

Note: The free tier includes 15 requests per minute and 1 million tokens per minute.

Your API key is stored locally and never shared. It's only used to make requests to Google's Gemini API.

About This Extension

Content Scanner uses Google's Gemini AI to analyze web page content in real-time. It can detect potentially malicious content, tracking scripts, AI-generated text, and misinformation. The analysis is performed locally using your API key, ensuring your privacy is maintained.
